IFRS S1/S2 Readiness Check 
Este projeto é uma ferramenta de autoavaliação (Readiness Check) desenvolvida em Python e Streamlit, projetada para auxiliar empresas e profissionais de auditoria/contabilidade a medir o nível de preparação para as novas normas globais de divulgação de sustentabilidade (IFRS S1 e S2).
  Funcionalidades
- Diagnóstico Setorial: Benchmarks customizados para os setores Financeiro, Energia, Indústria e Varejo.
- Estrutura Baseada no TCFD/IFRS: Avaliação dividida nos 4 pilares fundamentais:
  - Governança
  - Estratégia
  - Gestão de Riscos
  - Métricas e Metas
- Integração com COSO: Insights sobre riscos e controles baseados na estrutura COSO.
- Casos Reais: Exemplos práticos de mercado para ilustrar gaps e boas práticas.
- Visualização de Gaps: Gráficos interativos (Radar/Spider Chart) comparando a pontuação da empresa com o benchmark do setor.
  Tecnologias Utilizadas
- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/) (Interface Web)
- [Pandas](https://pandas.pydata.org/) (Manipulação de dados)
- [Plotly](https://plotly.com/) (Gráficos interativos)
